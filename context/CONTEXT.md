# Contexte — reseau de collaborations et similarites thematiques

Projet de theorie des graphes applique a des donnees open data reelles:

- CORDIS (projets + organisations, H2020 / Horizon Europe)
- OpenAlex (publications + concepts + grants)

Objectif:

1. Construire le reseau explicite (financement / co-participation)
2. Construire le reseau implicite (proximite thematique)
3. Mesurer le gap entre proximite scientifique et collaborations effectives

Stack:

- Python 3.11+
- pandas, networkx, requests
- sentence-transformers / faiss (optionnel, extension)

Langue:

- Reponses et documentation en francais
- Commentaires code en anglais
